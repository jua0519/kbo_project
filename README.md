# ⚾ team 08 - 롯데 자이언츠 팬덤 반응 분석

KBO 리그 경기 결과에 따른 팬들의 온라인 여론과 감정 변화를 추적하고 분석하는 프로젝트입니다. 부산의 뜨거운 야구 열정을 바탕으로, 경기 상황별 팬들의 반응 차이와 핵심 키워드를 도출합니다.

---

## 👥 팀원 및 역할 분담
* **김주아 (202641126): 데이터 수집 기획, 감성 분석 구현, 보고서 및 발표 자료 작성
* **조서원 (202641172): 크롤링 코드 계획 및 작성, 데이터 전처리 및 데이터 조인

---

## 🛠️ 사용 라이브러리 (Tech Stack)
프로젝트 구동을 위해 다음 파이썬 외부 라이브러리를 사용합니다.
* `requests` (v2.31.0): 네이버 스포츠 뉴스 및 검색 페이지 웹 데이터 요청
* `beautifulsoup4` (v4.12.3): HTML 파싱 및 필요한 텍스트 데이터 추출
* `csv`, `json`, `datetime`: 데이터 정제 및 CSV 파일 저장 (파이썬 기본 내장)
